# Orders Service

Backend service responsible for handling order requests.

---

## Requirements

- Python 3.10+

---

## Clone

```bash
git clone https://github.com/<your-username>/orders-service.git
cd orders-service
```

STEPS TO FOLLOW 

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

TO RUN

- uvicorn app:app --host 0.0.0.0 --port 8000
